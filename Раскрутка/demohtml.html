Какие основные преимущества скрипта World Translate?

  • Очень грамотный/хороший перевод текстов. При переводе страниц не нарушается разметка кода. 
  • Также при переводе не нарушается html код страницы. Можете открыть и посмотреть код на моем сайте.
  • Замена текста в alt, value, title, meta и т.д. Грамотная замена текста в ссылках.
  • Возможность указывать какие блоки не нужно переводить. Просто добавить класс notranslate в нужный блок (в редакторе) или в html коде.
  • Сайт переводится на 100% включая сторонни расширения. Легко встраивается в расширения использующию технологию ajax
  • Возможность определение языка пользовате и автоматическое открытие страницы на его языке.
  • Автоматическое определение исходного языка и последующий его перевод.
  • Легкая интеграция в любой сайт.

Как работает скрипт World Translate?

Мною разработан и написан на языке php специальный алгоритм, который позволяет корректно переводить любые html страницы. Для перевода слов я использовал предоставляемое API Googla При этом я утверждаю что на данный момент мой переводчик корректней переводит страницы чем любые другие существующие аналоги. Кратко работу скрипта можно описать следующим образом:

  • Вначале из страницы извлекаются все слова, которые необходимые перевести.
  • Извлеченные из страницы слова перевадятся с помощью API Googla
  • Все переведенные слова  записываются назад в html страницу.

При этом World Translate даже без кеширования работает очень быстро. Скорость работы скрипта зависит от количества текста, которые вы хотите перевести. В среднем генерация и перевод страницы занимает не более 10 секунд. Если же вы используете систему кеширования то генирация страницы может занимать вообще милисекунды. К примеру взгляните насколько быстро перевелись такие популярные сайты как:

Первевод www.yahoo.com на китайский языкПеревод www.joomla.org на немецкий языкПеревод www.rambler.ru на английский язык

Какие системные требования для работы скрипта World Translate?

  • Curl библиотека. Как правило уже установлена на большенстве хостингах, если не установлена то без проблем ставится.
  • PHP memory limit 32mb - 128mb. Чем больше, тем лучше.
  • Zend Optimizer - это бесплатный серверный модуль, который установлен практически на всех хостингах.

Для того что-бы узнать подходит ли текущая конфигурация вашего хостинга для скрипта World Translate создайте php-файл с одной строчкой.
